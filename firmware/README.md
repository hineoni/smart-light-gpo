# SmartLight Firmware (ESP-IDF)

Прошивка ESP32 для управления физическими компонентами системы Smart Light.

## Компоненты

- `wifi_manager` — подключение к Wi-Fi и восстановление соединения;
- `ble_provisioning` — первичная настройка устройства через BLE;
- `websocket_client` — heartbeat, команды управления и обмен данными с backend;
- `led_controller` — управление адресной светодиодной лентой;
- `servo_controller` — управление двумя сервоприводами;
- `uwb_positioning` — чтение расстояний с UWB-модуля MK8000 по UART;
- `config_storage` — хранение конфигурации устройства в NVS;
- `web_server` — локальные HTTP-эндпоинты устройства.

## Аппаратная конфигурация

В текущей конфигурации используются:

| Назначение | Интерфейс |
|------------|-----------|
| UWB MK8000 | UART1, TX GPIO18, RX GPIO19, 115200 baud |
| LED-лента | GPIO33 |
| Кнопка сброса | GPIO0 |

Роли UWB автоматически выбираются для известных идентификаторов устройств.
Для остальных устройств автоконфигурация MK8000 отключается, чтобы не
перезаписать неизвестную аппаратную настройку.

## Сборка

Установите ESP-IDF и активируйте его окружение, затем выполните:

```bash
idf.py set-target esp32
idf.py build
```

Для прошивки и просмотра журнала:

```bash
idf.py -p /dev/ttyUSB0 flash monitor
```

Порт необходимо заменить на порт подключённой ESP32. Параметры сборки находятся
в `sdkconfig.defaults`, а таблица разделов — в `partitions.csv`.
